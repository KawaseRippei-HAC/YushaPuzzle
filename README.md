# 勇者と魔王をつなぐ道

専門学校の2年次にチーム制作で開発した、「3Dパズル × ダンジョンメーカー」を掛け合わせたゲームです。

## 🎮 ゲーム概要
- **ジャンル:** 3Dパズル × ダンジョンメーカー
- **開発環境:** C++ / GSLIB
- **開発時期:** 2年生後期 10月～1月 約370時間
- **チームの構成:** プランナー 1名 プログラマー 3名
- **動作プラットフォーム:** Windows

## 💻 担当箇所（ソースコード）
本リポジトリにはプロジェクト全体を格納しています。
私が主に担当・実装したのは以下の機能（ファイル）です。

* `Events/` : イベント関連の進行・制御処理全般
* `Systems/AssetLoader.cpp, .h` : アセット（モデルや画像等）の外部ファイル読み込み・管理クラス
* **【バトル関連システム】**
  * `Systems/BattleSystem.cpp, .h` : バトルの進行や判定を統括するシステムクラス
  * `Monsters/` : 敵キャラクターの挙動・ステータス等、バトル側の処理

## ✨ アピールポイント・苦労した点
**未経験ジャンルにおける設計への挑戦とチーム連携**
 - これまでアクションゲームのベースしか経験がなかったため、パズルゲーム特有のシステム設計に非常に苦労しました。しかし、実装に入る前にチーム全体でしっかりと時間を取って要件定義と設計のすり合わせを行ったことで、結果的に大きな破綻なく完成まで導くことができました。
**再利用性を意識した汎用的なクラス設計**
 - チーム全体の開発効率を上げるため、一部の機能に依存しない「全体的に応用が利く設計（コンポーネント化）」を意識しました。作成したシステムを他の場面でも使い回せるように構築したことで、実装の手間が省け、プロジェクト全体の作業効率向上に貢献しました。
